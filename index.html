Sistema de Carga Solar Fotovoltaico

Colegio Calatrava - Proyecto STEAM+

Integrantes: Gabriel Posada, Damian Rojas, Samuel Ibañez, Cristobal Cuta

¿Por qué este proyecto?

Este proyecto busca aprovechar la energía solar para alimentar un puesto de carga de dispositivos electrónicos dentro de una de las aulas del colegio. Queremos promover el uso de energías limpias y enseñar su funcionamiento a través de una instalación real y educativa.
